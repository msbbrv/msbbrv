Що таке Git?Git — розподілена система керування версіями файлів
 та спільної роботи. Git є однією з найефективніших,
 надійних і високопродуктивних систем керування версіями, що надає гнучкі
 засоби нелінійної розробки, що базуються на відгалуженні і злитті гілок.
В якому статусі можуть перебувати файли при роботі з Git?- Untracked, Unmodified, Modified, Staged
Для чого використовується команда add?-  приймає шлях файлу або директорії
Для чого використовується команда status?- щоб дізнатися статус файлу 
Для чого використовується команда commit?- для створення коміту
для чого використовується команда init? - для включення Git з данного репозиторію
Що необхідно писати в коментарях для комітів?- номерування коміту?
як переглянути список комітів?- git log
ля чого використовується команда checkout?- поверненя файлу до попереднього стану
